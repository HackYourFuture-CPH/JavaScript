Hello DOM

Hi, my name is Document Object Model (DOM). I am a programming interface for HTML documents. I represent the page so that programs can change my structure, style and content. I represent the document as nodes and objects. That way, programming languages can connect to the page.

A Web page is a document. This document can be either displayed in the browser window or as the HTML source. But it is the same document in both cases. I, the Document Object Model (DOM), represent that same document so it can be manipulated. I am an object-oriented representation of the web page, which can be modified with a scripting language such as JavaScript.
